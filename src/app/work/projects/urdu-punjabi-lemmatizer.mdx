## Overview

Developed a deep-learning bilingual lemmatizer for Urdu and Punjabi to improve downstream NLP tasks such as information retrieval, machine translation, and text classification.

## Repo & Install

- Repository: https://github.com/malaikabasharat004/urdu-Punjabi-lemmatizer
- Install via pip:

```bash
pip install urdupunjabilemma
```

## Dataset

- Public dataset: https://www.kaggle.com/datasets/malaikabasharat/urdu-punjabi-merged-dataset/data
- Dataset stats: ~60k Punjabi tokens, ~40k Urdu tokens (see Kaggle page for details)

## Demo & Usage

Include a short code example to show how to use the package in Python:

```python
from urdupunjabilemma import Lemmatizer

lemmatizer = Lemmatizer()
print(lemmatizer.lemmatize("کتابیں"))  # example output
```
